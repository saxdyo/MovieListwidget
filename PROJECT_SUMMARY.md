### 1. 🔧 核心爬虫脚本
- **`scripts/get_tmdb_data.py`**: 增强版TMDB数据爬虫

### 4. 📚 完整文档体系

## 🚀 快速使用指南

### 1. 环境配置
```bash
pip install -r requirements.txt
```

### 2. API配置
```bash
export TMDB_API_KEY="your_api_key_here"
```

### 3. 运行脚本
